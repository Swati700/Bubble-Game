# 🫧 Bubble Game

A fun and interactive Bubble Game built using **HTML**, **CSS**, and **JavaScript**.

The goal is simple: click the bubble that matches the **Hit Number** before the timer runs out. Every correct click increases your score and generates a new target number.

---

## 🚀 Features

- 🎯 Random target (Hit) number
- ⏱️ 60-second countdown timer
- 💯 Live score tracking
- 🔄 New bubbles generated after every correct hit
- 🎲 Random numbers on every bubble
- 🏁 Game Over screen with final score

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## ▶️ How to Play

1. Open the game in your browser.
2. Check the **Hit** number displayed at the top.
3. Click the bubble containing the same number.
4. Earn **10 points** for every correct click.
5. Continue until the timer reaches **0**.
6. Your final score is displayed when the game ends.
